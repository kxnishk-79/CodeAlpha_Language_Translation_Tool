========================================
OMNIVOX — LANGUAGE TRANSLATION TOOL
========================================

📌 Project Overview:
OmniVox is a modern, real-time language translation web application designed to make communication seamless across languages. It provides a clean, interactive interface with support for 50+ languages, along with additional usability features like text-to-speech.

----------------------------------------
🚀 Features:
----------------------------------------
- Real-time language translation
- Support for 50+ global languages
- Clean and modern UI/UX design
- Auto language selection interface
- Text-to-speech functionality
- Copy translated text
- Smooth animations and responsive layout
- Dark/Light theme support

----------------------------------------
🛠️ Technologies Used:
----------------------------------------
- HTML5 (Structure)
- CSS3 (Advanced UI, animations, theming)
- JavaScript (Logic & API handling)
- MyMemory Translation API (No API key required)
- Google Fonts (Inter, JetBrains Mono)

----------------------------------------
⚙️ How It Works:
----------------------------------------
1. User enters text in the input field
2. Selects source and target languages
3. Application sends request to MyMemory API
4. Receives translated response
5. Displays translated text instantly
6. Optional features: Copy or listen via text-to-speech

----------------------------------------
📂 Project Structure:
----------------------------------------
- index.html     → Main UI structure
- style.css      → Styling, themes, animations
- script.js      → Application logic & API integration
- package.json   → Dev server setup

----------------------------------------
💡 Key Learnings:
----------------------------------------
- Handling API requests using JavaScript
- Working with asynchronous operations
- Designing modern UI with CSS variables
- Managing large datasets (language list)
- Enhancing UX with animations and features

----------------------------------------
⚠️ Notes:
----------------------------------------
- Requires internet connection
- Translation accuracy depends on API
- Free API may have limitations
