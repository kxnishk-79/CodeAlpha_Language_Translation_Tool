## 📌 Overview
**OmniVox** is a sleek and modern language translation web application that enables users to translate text across 50+ languages in real-time.

Designed with a strong focus on **UI/UX**, performance, and usability, it delivers a smooth and interactive translation experience.

---

## ✨ Features

- 🌐 Supports **50+ Languages**
- ⚡ Real-time Translation
- 🎨 Clean & Modern UI Design
- 🌓 Dark / Light Theme Support
- 🔊 Text-to-Speech Output
- 📋 Copy to Clipboard
- 📱 Responsive Layout
- ⚠️ Error Handling & Feedback
